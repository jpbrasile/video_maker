Avant de commencer, assurez-vous d'avoir Python installé sur votre système. Ensuite, installez les bibliothèques nécessaires en utilisant pip, comme Python-dotenv, OpenAI et d'autres. Enfin, créez un fichier .env dans le même répertoire que votre script pour y stocker vos clés API Anthropic et OpenAI. Une fois cette configuration de base effectuée, vous serez prêt à passer à l'étape suivante.
