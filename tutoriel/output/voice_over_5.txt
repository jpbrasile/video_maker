Pour chaque diapositive, plusieurs éléments sont générés automatiquement. D'abord, le code HTML pour afficher le contenu. Ensuite, un texte de voix off est créé pour expliquer la diapositive. Une image illustrative est également produite pour la rendre plus attrayante visuellement. Enfin, un fichier audio est généré à partir du texte de narration pour permettre une présentation sonore.
