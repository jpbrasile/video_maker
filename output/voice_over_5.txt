Le Dockerfile définit l'environnement de conteneurisation pour notre API mathématique. Il spécifie la base système d'exploitation, installe les dépendances requises et copie le code source. Différentes options sont possibles, comme choisir une image de base légère ou une version spécifique des dépendances. L'objectif est de créer un conteneur reproductible et optimisé pour exécuter efficacement notre application.
