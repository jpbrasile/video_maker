Exemple de Code Python : Hello, World!

Le code ci-dessous définit une fonction print_hello_world qui imprime "Hello, World!".

La fonction main appelle cette fonction, et le script vérifie si le nom du module est __main__ pour exécuter main lorsqu'il est exécuté en tant que programme principal.


def print_hello_world():
    print("Hello, World!")

def main():
    print_hello_world()

if __name__ == "__main__":
    main()
